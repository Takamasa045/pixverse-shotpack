# Brief: Signature Product Launch

## Title
Signature Product Launch - Atmospheric Reveal

## Objective
- Goal: 新しいプロダクトの存在感と質感を短尺映像で伝え、ブランドの第一印象を整える
- Desired viewer action: LP で詳細を見る / 製品ページへ進む
- Core takeaway: 静かで上質な世界観の中に、手に取りたくなる主役プロダクトがある

## Audience
- Primary audience: デザイン感度の高いオンライン視聴者、ブランド担当者、クリエイティブチーム
- Platform / placement: LP hero / SNS 広告 / Short-form video
- Context of viewing: 新商品公開、ブランド紹介、キャンペーン導線

## Tone
- Mood: calm / premium / tactile
- Color direction: warm neutrals + soft charcoal + restrained highlight
- Tempo: slow
- Texture / finish: cinematic, shallow depth of field, soft bloom, fine grain

## Workflow Preference
- Preferred workflow: i2v
- Image reference policy: generate
- Credit budget: 50

## Target Duration
25-30 seconds

## Shot Count
6 shots

## Key Visuals
1. 空間全体を見せる導入ショット
2. 素材感の分かるディテール
3. 主役プロダクトのヒーローショット
4. 手元の操作または設置シーン
5. 使用感や動きを示す中盤ショット
6. 余韻を残すクロージングフレーム

## Style Reference
- Premium editorial product film - 落ち着いたライティングと余白
- Natural-light still-life cinema - 素材感を丁寧に見せる構図

## Constraints
- Must include: 主役プロダクト、素材ディテール、落ち着いた空間演出
- Must avoid: 過剰なエフェクト、読み取れる文字、派手なネオン表現
- Faces / people: 手元またはシルエットまで
- Text / logo policy: 文字とロゴは Remotion で後入れ
- Other production rules: ラベルや UI の可読文字は画に入れない

## Delivery
- Primary: 16:9 / 1080p / 30fps
- Secondary: 9:16 / 1080p / 30fps
- Format: MP4 + Remotion manifest.json

## Gate Notes
- Gate 1 で必ず確認したいこと: 世界観の静けさと shot 数
- Gate 2 で NG とみなす条件: 質感が人工的すぎる、主役が小さすぎる、手元が破綻している

## Optional Notes
- Audio / music direction: minimal ambient + soft percussive pulse
- Camera preference: dolly / tracking を主軸、zoom は使いすぎない
- Priority motif: silhouette / texture / hero product / hand interaction
- Post-process preference: 最後の shot だけ軽い extend を許容
